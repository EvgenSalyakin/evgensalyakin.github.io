Description: The push() method can append one or more elements to the end of an array. The pop() method pulls the last element off of the given array and returns it. The unshift() method is like the push() method, only it works at the beginning of the array. The unshift() method can prepend one or more elements to the beginning of an array. The shift() method is like the pop() method, only it works at the beginning of the array. The shift() method pulls the first element off of the given array and returns it.

Example:
item = arr.shift();
arr.push(item);
